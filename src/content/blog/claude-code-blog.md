## 事情的起因

作为一个准备一人公司的码农，我一直想搭一个个人博客。需求很简单：写文章、挂广告、支持中英文。

以前这种事，我会花一两天搭框架、调样式、配部署。但今天，我决定试试让 **Claude Code** 来干这个活。

结果？**从零到上线，我全程没写一行代码。**

## Claude Code 都干了啥？

### 1. 项目搭建

我只说了一句"帮我建一个个人博客"，Claude Code 就：

- 选了 **Astro** 作为框架（静态站点生成器，天生适合博客）
- 初始化项目、安装依赖
- 创建了完整的项目结构

### 2. 一个配置文件搞定一切

它帮我创建了一个 `src/config.ts`，把所有可配置项集中在一起：

```typescript
export const SITE = {
  title: { en: "Solopreneur", zh: "独立营业中" },
  author: { name: { en: "Eric Gao", zh: "德州小高" } },
  analytics: { googleAnalyticsId: "" },
  ads: { googleAdsenseId: "" },
  theme: { primaryColor: "#2563eb" },
  // ...
};
```

以后想改啥？告诉 Claude Code 一句话就行："把主色调改成红色"、"加上我的 GitHub 链接"。

### 3. 中英文双语

博客自动支持中文和英文，中文是默认语言，英文在 `/en/` 路径下。导航栏有语言切换按钮，一键切换。

更狠的是，它还写了一个 **AI 翻译脚本**——我只需要写中文文章，运行一个命令就能自动翻译成英文：

```bash
node scripts/translate.mjs src/content/blog/my-post.md
```

一篇文章，两个语言版本，全自动。

### 4. 广告和分析

Google AdSense 和 Google Analytics 的代码已经预埋在模板里了。我只需要在配置文件里填上 ID，广告就自动出现在文章页面。

### 5. 免费部署到 GitHub Pages

这是最爽的部分：

- Claude Code 写了 GitHub Actions workflow
- 我每次 `git push`，网站就自动构建和部署
- **完全免费**，GitHub Pages 对公开仓库不收钱
- 自定义域名？配个 DNS 就搞定了

整个过程，我做的最"技术"的操作就是在 Cloudflare 上改了一下 DNS 记录。😂

## 下一步：自动选题

说实话，现在这个系统还不是全自动的。目前的流程是这样的：

1. **我** 提出文章方向和大纲
2. **Claude Code** 帮我生成内容和翻译
3. **我** review 内容，修改后发布

瓶颈在第一步——选题和方向还是得我自己想。

所以接下来我要做的是：**加一个自动选题功能。** 让 AI 根据我的博客定位、热门趋势和 SEO 数据，自动推荐选题方向，甚至直接生成初稿。

到那时候，我的工作流就变成了：

1. AI 自动推荐选题 + 生成草稿
2. **我只需要 review 和发布**

从"我写博客"变成"AI 写博客，我把关质量"。这才是一个 Solopreneur 应该有的姿态——**把时间花在决策上，而不是执行上。**
